Voici le projet d'IO2 de Eva Herson MI2 et Kylian Punel MI1.

Pour pouvoir faire fonctionner MySQL et donc le site,
allez sur le fichier base_donnee.php et rentrez vos informations pour vous paramètres d'utilisateur.

Une base de données SQL est fournie dans le dossier,
n'hésitez pas à l'importer.
Elle contient toutes les tables utiles au fonctionnement du site ainsi que des utilisateurs déjà enregistrés.

Pour rénitialiser le site,
il suffit de vider toutes les tables sauf potentiellement celle avec les universités.
De plus, il faut supprimer les images des dossiers publications et pdp.
Pour ce dernier, veillez à garder personne0.png qui est la photo de profil par défault.

Pour remplir les tables SQL, tout peut se faire depuis le site.
Sauf la colonne administrateur de la table utilisateur qui doit être modifiée directement avec une ligne de commande SQL.
Nous considérons que le rôle d'administrateur est important, c'est pour cela que l'on peut le modifer seulement de cette manière.

Le site est maintenant prêt à l'emploi, merci de l'utiliser.
